**Idee / Idea:** Ein `/review`-Befehl, der einmal pro Woche alle in den letzten 7 Tagen erfassten Lessons sammelt, sie nach Domain gruppiert und eine kurze bilinguale Zusammenfassung mit den am häufigsten ausgelösten Regeln ausgibt. / A `/review` command that, once a week, collects every lesson captured in the last 7 days, groups them by domain, and prints a short bilingual summary highlighting the most-fired rules.

## Nächster Schritt / Next step
- Definiere das Ausgabeformat an einem einzigen Beispiel: Header mit Datumsspanne, dann eine Liste der Lessons mit `[[wikilinks]]`. / Define the output format from a single example: a header with the date range, then a list of lessons with `[[wikilinks]]`.
